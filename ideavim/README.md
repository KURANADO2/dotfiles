## Plugin Prepare

IDEA Install the following plug-ins:

- IdeaVim
- IdeaVim-EasyMotion
- IdeaVimExtension
- AceJump

## IDEA Action List

You can use the `:actionlist` command in vim to list all IDEA actions, then you can configure the vim shortcut according to your needs.
